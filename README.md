# WEATHER APP

Weather app is basic project to show the current of places by using [Open Weather Map API](https://openweathermap.org/current).

## Built Using
* **Html.**
* **CSS**
* **EJS.** 
* **Javascript.**
* **Node.js**
* **Express**

## Hosting
